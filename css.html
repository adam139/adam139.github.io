移动应用优先

在 Bootstrap 2系列版本中，我们增加了可选的支持移动应用的部分。在Bootstrap 3,我们以移动应用友好的原则彻底重写整个项目，不仅仅是添加可选的移动应用样式 。 事实上， Bootstrap3 是移动优先的，移动应用优先的原则贯彻在整个项目当中。

为了保障实现触摸式缩放， 添加 viewport meta tag到 <head>.

{% highlight html %} {% endhighlight %}

你通过添加user-scalable=no 到 viewport meta tag中，我们能够取消对移动设备缩放功能的 支持。这样做结果意味着用户仅仅能够滚动浏览，感觉站点就像一个本地应用。总之，我们不推荐某个站点都这么做，使用时，请注意！

网格系统

Bootstrap 包括一个自适应，移动优先的、浮动的12格网格系统 。它包含 一些预定义的css classes 来实现容易的排版和强大的功能。 mixins for generating more semantic layouts.

Media queries

我们用如下的media queries在我们的 LESS文件中，以便在我们的网格系统中创建关键的载入点。

{% highlight css %} /* 非常小的设备 (phones, less than 768px) */ /* 没有媒体查询规则，这是在 Bootstrap3 中默认的 */ /* 小设备 (tablets, 768px and up) */ @media (min-width: @screen-sm-min) { ... } /* 中等设备(desktops, 992px and up) */ @media (min-width: @screen-md-min) { ... } /* 大设备 (large desktops, 1200px and up) */ @media (min-width: @screen-lg-min) { ... } {% endhighlight %}

我们有时也扩展媒体查询包括一个 max-width 以设置CSS被应用到指定的最小、最大的宽度空间的所有设备。

{% highlight css %} @media (max-width: @screen-xs-max) { ... } @media (min-width: @screen-sm-min) and (max-width: @screen-sm-max) { ... } @media (min-width: @screen-md-min) and (max-width: @screen-md-max) { ... } @media (min-width: @screen-lg-min) { ... } {% endhighlight %}

采用LESS系统并引入变量

In addition to prebuilt grid classes for fast layouts, Bootstrap includes LESS variables and mixins for quickly generating your own simple, semantic layouts.

变量

Variables determine the number of columns, the gutter width, and the media query point at which to begin floating columns. We use these to generate the predefined grid classes documented above, as well as for the custom mixins listed below.

{% highlight css %} @grid-columns: 12; @grid-gutter-width: 30px; @grid-float-breakpoint: 768px; {% endhighlight %}

混合

混合可以将一个定义好的class A轻松的引入到另一个class B中，从而简单实现class B继承class A中的所有属性。我们还可以带参数地调用，就像使用函数一样。在Bootstrap中，混合被用来结合网格变量为独立的网格生成更具语义性CSS（LESS）。

{% highlight css %} // Creates a wrapper for a series of columns .make-row(@gutter: @grid-gutter-width) { // Then clear the floated columns .clearfix(); @media (min-width: @screen-sm-min) { margin-left: (@gutter / -2); margin-right: (@gutter / -2); } // Negative margin nested rows out to align the content of columns .row { margin-left: (@gutter / -2); margin-right: (@gutter / -2); } } // 生成特别小的格子 .make-xs-column(@columns; @gutter: @grid-gutter-width) { position: relative; //当内容为空时，阻止格子被折叠 min-height: 1px; //内间距通过padding实现 padding-left: (@gutter / 2); padding-right: (@gutter / 2); //基于有效格子数目计算格子的宽度（百分比） @media (min-width: @grid-float-breakpoint) { float: left; width: percentage((@columns / @grid-columns)); } } // 生成小的格子 .make-sm-column(@columns; @gutter: @grid-gutter-width) { position: relative; //当内容为空时，阻止格子被折叠 min-height: 1px; //内间距通过padding实现 padding-left: (@gutter / 2); padding-right: (@gutter / 2); //基于有效格子数目计算格子的宽度（百分比） @media (min-width: @screen-sm-min) { float: left; width: percentage((@columns / @grid-columns)); } } // Generate the small column offsets .make-sm-column-offset(@columns) { @media (min-width: @screen-sm-min) { margin-left: percentage((@columns / @grid-columns)); } } .make-sm-column-push(@columns) { @media (min-width: @screen-sm-min) { left: percentage((@columns / @grid-columns)); } } .make-sm-column-pull(@columns) { @media (min-width: @screen-sm-min) { right: percentage((@columns / @grid-columns)); } } // Generate the medium columns .make-md-column(@columns; @gutter: @grid-gutter-width) { position: relative; //当内容为空时，阻止格子被折叠 min-height: 1px; //内间距通过padding实现 padding-left: (@gutter / 2); padding-right: (@gutter / 2); //基于有效格子数目计算格子的宽度（百分比） @media (min-width: @screen-md-min) { float: left; width: percentage((@columns / @grid-columns)); } } // 生成中等大小格子的偏移 .make-md-column-offset(@columns) { @media (min-width: @screen-md-min) { margin-left: percentage((@columns / @grid-columns)); } } .make-md-column-push(@columns) { @media (min-width: @screen-md-min) { left: percentage((@columns / @grid-columns)); } } .make-md-column-pull(@columns) { @media (min-width: @screen-md-min) { right: percentage((@columns / @grid-columns)); } } // 生成大格子 .make-lg-column(@columns; @gutter: @grid-gutter-width) { position: relative; //当内容为空时，阻止格子被折叠 min-height: 1px; //内间距通过padding实现 padding-left: (@gutter / 2); padding-right: (@gutter / 2); //基于有效格子数目计算格子的宽度（百分比） @media (min-width: @screen-lg-min) { float: left; width: percentage((@columns / @grid-columns)); } } // Generate the large column offsets .make-lg-column-offset(@columns) { @media (min-width: @screen-lg-min) { margin-left: percentage((@columns / @grid-columns)); } } .make-lg-column-push(@columns) { @media (min-width: @screen-lg-min) { left: percentage((@columns / @grid-columns)); } } .make-lg-column-pull(@columns) { @media (min-width: @screen-lg-min) { right: percentage((@columns / @grid-columns)); } } {% endhighlight %}
